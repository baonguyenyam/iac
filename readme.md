# Features

- Build on Bootstrap Frameworks
- Using SVG Images 
- Mobile friendly
- Tested on:
- - Mobile: iPhone X (375 x 812)
- - Tablet: iPad (768 x 1024)
- - Tablet: iPad Pro (1024 x 1366)
- - Laptop: MDPI screen (1280 x 800)
- - Laptop: HiDPI screen (1440 x 900)
- - Tivi: 4K screen (1920 x 1080)

# Setup

> npm i 

Install Components 

> bower update 

# Run dev

> npm start

# Production 

> npm run pro
